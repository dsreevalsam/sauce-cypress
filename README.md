# sauce-cypress

This project contains automated tests for www.saucedemo.com

Prerequisite: latest node.js installed

clone the repo using "git clone https://github.com/dsreevalsam/sauce-cypress.git"
goto project root and run "npm install" (installs cypress and other dependencies"

Run Tests
---------
Run the command: "npm run cypress:open" to open cypress gui on a browser
Select the tests to run

Alternatively,

Run the command: "npm run cy:run" to run the test on commandline.
